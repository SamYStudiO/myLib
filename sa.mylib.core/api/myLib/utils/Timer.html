PackagemyLib.utils
Classpublic class Timer
InheritanceTimer flash.events.EventDispatcher

Timer class is equivalent to flash.utils.Timer class but is syncronize with system clock in order to make easier countdown. Even if processor is slow down, each intervals are always the same.
