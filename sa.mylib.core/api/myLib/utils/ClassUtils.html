describeType()method 
public static function describeType(o:*):XML

describetype is the same as flash.utils.describeType but use an internal cache in order to optimize multiple call to the same object since this can be expansive. The bad thing is cache increase memory consumption, if you have any memory problem you should use clearDescribeTypeCache method to flush cache.
