Tween()Constructor
public function Tween(obj:Object, prop:* = null, ease:* = null, begin:* = null, finish:* = null, duration:uint = 0, durationAsMilliSeconds:Boolean = false, startDelay:uint = 0)

Build a new Tween instance.

Parameters
obj:Object — The object which contains properties to tween.
 
prop:* (default = null) — The property or Array of properties to tween.
 
ease:* (default = null) — The ease function or Array of ease Function used for tweening.
 
begin:* (default = null) — The begin value or Array of begin values to used with property (if NaN current property value is used).
 
finish:* (default = null) — The finish value or Array of finish values to used with property.
 
duration:uint (default = 0) — The tween duration expressed in frames or milliseconds depending next arguments.
 
durationAsMilliSeconds:Boolean (default = false) — Defined how is expressed duration (frames or milliseconds).
 
startDelay:uint (default = 0) — A delay in milliseconds to wait before tween start. if startDelay is 0 tween start immediatly.
