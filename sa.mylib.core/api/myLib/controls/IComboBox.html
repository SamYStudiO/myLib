dropdownWidthTypeproperty 
dropdownWidthType:String

Get or set the dropdown width type as defined in ComboBoxDropdownWidthType constants. NONE value make dropdown list resized according ComboBox width. PERCENTAGE value make dropdown list resized in percentage in association with dropdownWidth property. ARROW_BUTTON value make dropdown list resized according ComboBox width - ComboBox arrow button size. AUTOSIZE value make dropdown list resized to make completly visible all cells.

The default value is ComboBoxDropdownWidthType.NONE.
