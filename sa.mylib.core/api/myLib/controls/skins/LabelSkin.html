Classpublic class LabelSkin
InheritanceLabelSkin ASkin flash.events.EventDispatcher
Implements ILabelSkin
Subclasses TextInputSkin

LabelSkin is the default skin for Label component. Like all other components, default skin use string definition, BitmapData object or external URL to build assets. To use other values or for complexe skin make your own ILabelSkin implementation.
