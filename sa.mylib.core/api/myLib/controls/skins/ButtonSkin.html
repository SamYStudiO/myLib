PackagemyLib.controls.skins
Classpublic class ButtonSkin
InheritanceButtonSkin ASkin flash.events.EventDispatcher
Implements IButtonSkin
Subclasses CheckBoxSkin, RadioButtonSkin

ButtonSkin is the default skin for Button component. Like all other components, default skin use string definition, BitmapData object or external URL to build assets. To use other values or for complexe skin make your own IButtonSkin implementation.
