Classpublic class ColorPickerSkin
InheritanceColorPickerSkin ASkin flash.events.EventDispatcher
Implements IColorPickerSkin

ColorPickerSkin is the default skin for ColorPicker component. Like all other components, default skin use string definition, BitmapData object or external URL to build assets. To use other values or for complexe skin make your own IColorPickerSkin implementation.
