Classpublic class CheckBoxSkin
InheritanceCheckBoxSkin ButtonSkin ASkin flash.events.EventDispatcher

CheckBoxSkin is the default skin for CheckBox component. Like all other components, default skin use string definition, BitmapData object or external URL to build assets. To use other values or for complexe skin make your own IButtonSkin implementation.
