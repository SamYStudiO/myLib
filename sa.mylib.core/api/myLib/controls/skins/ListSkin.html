Classpublic class ListSkin
InheritanceListSkin ASkin flash.events.EventDispatcher
Implements IListSkin

ListSkin is the default skin for List component. Like all other components, default skin use string definition, BitmapData object or external URL to build assets. To use other values or for complexe skin make your own IListSkin implementation.
