Classpublic class TextInputSkin
InheritanceTextInputSkin LabelSkin ASkin flash.events.EventDispatcher
Implements ITextInputSkin
Subclasses TextAreaSkin

TextInputSkin is the default skin for TextInput component. Like all other components, default skin use string definition, BitmapData object or external URL to build assets. To use other values or for complexe skin make your own ITextInputSkin implementation.
