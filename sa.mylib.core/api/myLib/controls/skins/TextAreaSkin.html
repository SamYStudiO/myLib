Classpublic class TextAreaSkin
InheritanceTextAreaSkin TextInputSkin LabelSkin ASkin flash.events.EventDispatcher
Implements ITextAreaSkin

TextAreaSkin is the default skin for TextArea component. Like all other components, default skin use string definition, BitmapData object or external URL to build assets. To use other values or for complexe skin make your own ITextAreaSkin implementation.
