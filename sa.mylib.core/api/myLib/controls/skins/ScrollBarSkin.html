Classpublic class ScrollBarSkin
InheritanceScrollBarSkin ASkin flash.events.EventDispatcher
Implements IScrollBarSkin

ScrollBarSkin is the default skin for ScrollBar component. Like all other components, default skin use string definition, BitmapData object or external URL to build assets. To use other values or for complexe skin make your own IScrollBarSkin implementation.
