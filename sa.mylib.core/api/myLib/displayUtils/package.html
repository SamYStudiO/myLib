USE_PIXEL_SNAPPINGproperty
public var USE_PIXEL_SNAPPING:Boolean

A Boolean value that indicates whether actions in myLib, which change displayObject coordinates or size, are rounded to snap nearest pixel. If displayObject is move or resize before this property is set to true, its coordinates and size will not change.
