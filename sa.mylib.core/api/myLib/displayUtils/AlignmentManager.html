public static function align(o:DisplayObject, alignmentPoint:String, targetCoordinateSpace:DisplayObjectContainer = null, useGetBounds:Boolean = false):void

Align DisplayObject to the specified with the specfied alignment point and coordinate space.

Parameters

o:DisplayObject — The DisplayObject to align.
 
alignmentPoint:String — An alignment point as defined in AlignmentPoint constants.
 
targetCoordinateSpace:DisplayObjectContainer (default = null) — The displayObjectContainer that defined the coordinate space to use. If target is null current DisplayObject coordinate space ( its parent ) is used.
 
useGetBounds:Boolean (default = false) — By default object is align using DisplayObject.getRect() method, set useGetBounds to true to use DisplayObect.getBounds() insteadof DisplayObject.getRect().

public static function getAlignmentPoint(o:DisplayObject, alignmentPoint:String, targetCoordinateSpace:DisplayObjectContainer = null, fromX:Number, fromY:Number, useGetBounds:Boolean = false):Point

Get coordinates from a DisplayObject as a Point that reflects the specified alignment point and coordinate space.

Parameters

o:DisplayObject — The DisplayObject from which retrieve coordinates informations.
 
alignmentPoint:String — An alignment point as defined in AlignmentPoint constants.
 
targetCoordinateSpace:DisplayObjectContainer (default = null) — The displayObjectContainer that defined the coordinate space to use. If target is null current DisplayObject coordinate space ( its parent ) is used.
 
fromX:Number (default = NaN) — The DisplayObject x coordinate to use for retrieving Point. Leave empty or NaN to retrieve Point with actual DisplayObject x coordinate.
 
fromY:Number (default = NaN) — The DisplayObject y coordinate to use for retrieving Point. Leave empty or NaN to retrieve Point with actual DisplayObject y coordinate.
 
useGetBounds:Boolean (default = false) — By default point is retrieve using DisplayObject.getRect() method, set useGetBounds to true to use DisplayObect.getBounds() insteadof DisplayObject.getRect().

Returns
Point — A point with coordinates that reflects the specified alignment point and coordinate space.
