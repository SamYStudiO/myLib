Classpublic class StreamClient
InheritanceStreamClient flash.events.EventDispatcher
Implements IStreamClient

StreamClient class is useful with media which use a netStream object to specified all callback within this netStream. Generaly only onMetaData callback is necessary so you do not need to do anything. If you have specific callback with your netStream media inherit this class to add your own and add an instance to your StreamMedia object with streamClient property.
