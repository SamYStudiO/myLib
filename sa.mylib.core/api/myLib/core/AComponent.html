move()method 
override public function move(x:Number, y:Number, alignmentPoint:String = TL, targetCoordinateSpace:DisplayObjectContainer = null):void

Move asset to the specified x and y position according alignmentPoint and targetCoordinateSpace. By default asset is moved with top left alignment point and parent coordinate space. You can use alignmentPoint and targetCoordinateSpace arguments for advanced layout.

Parameters

x:Number — the x coordinate in pixel.
 
y:Number — the y corrdinate in pixel.
 
alignmentPoint:String (default = TL) — A alignment point as defined in AlignmentPoint constants that defined the refered point where move asset.
 
targetCoordinateSpace:DisplayObjectContainer (default = null) — The coordinate space to used. By default coordinates are affected within asset parent DisplayObjectContainer. Use this argument to move asset as it would be within the specific target.
