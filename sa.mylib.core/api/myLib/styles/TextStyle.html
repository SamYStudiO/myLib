PackagemyLib.styles
Classpublic dynamic class TextStyle
InheritanceTextStyle Object

The TextStyle class contains all style informations for a TextField. When a TextStyle is associated with a TextField all its properties are merged with that TextField. TextStyle is dynamic so you can add any properties that match a TextField property.
