Interfacepublic interface IField extends flash.events.IEventDispatcher
Implementors AFieldComponent, Button, ButtonGroup, ColorPicker, ComboBox, FieldGroup, List, Slider, TextInput

IField defines all properties and methods necessary with an input field. All input component (TextInput, TextArea, ComboBox,...) implement IField.
