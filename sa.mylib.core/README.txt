	INSTALLATION ***************************************************************
	
		- [Developer/Designer] Run bin/myLib.mxp or copy manually "bin/myLib.fla" to your flash component directory (Windows : "C:\Program Files\Adobe\Adobe Flash CS3\[lang]\Configuration\Components").
		- [Developer/Designer] Restart Flash IDE or refresh your component panel if Flash is already running.
		- [Developer] CS4/CS5/Flex users should add bin/myLib.swc to classpath, CS3 users should add "src" folder to Flash classpath.
		
		To open components panel go to > Window > components (ctrl + f7).
		If myLib components do not displayed try :
			- press components panel top right context button > Reload Components 
			- otherwise myLib.fla is maybe not installed where it should be, go to [programs_path]\Adobe\[flash], in each languages folders try to find myLib.fla in components folder and copy it to others languages.
		
		That's it!
